Quick Installation

# 1. Download and extract
cd ~/.openclaw/skills
unzip minimax-vision-search.zip

# 2. Install dependencies
cd minimax-vision-search/api
npm install

# 3. Configure API key
node ../scripts/setup-config.js --api-key "YOUR_API_KEY"

# 4. Start using the skill!
node ../scripts/web-search.js --query "test search"
